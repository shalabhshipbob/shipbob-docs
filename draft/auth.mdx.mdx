### How to get your API Key?

1. [Log in](https://web.shipbob.com/app/merchant/) to your ShipBob account. (If you don’t have it, click ***Create account*** at the bottom of the login page to create one for FREE.)

2. Visit [API Tokens](https://web.shipbob.com/app/merchant/#/Integrations/token-management).

3. Click ***Generate new token*** and follow the given instructions to generate your API key.

## 1. API Key

The API Key method is a more straightforward authentication method that only verifies whether the API Key value is correct or not.

### Required Headers

| **Header Name** | **Description**                                 | Example                                |
| --------------- | ----------------------------------------------- | -------------------------------------- |
| as-api-key      | The API key retrieved from the developer portal | asat\_a76eb8a2af3547df8a3c0c480f7f23d5 |
