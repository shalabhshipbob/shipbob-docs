This documentation provides an overview and API request examples for getting product inventory history events from ShipBob.

## Base Setup

The base URL for all inventory history requests should be:

* Production base URL: [`https://api.shipbob.com/2.0/`](https://api.shipbob.com/2.0/)

* Sandbox base URL: [`https://sandbox-api.shipbob.com/2.0/`](https://sandbox-api.shipbob.com/2.0/)

&#x20;

## Get Inventory History&#x20;

**Endpoint**: `GET /inventory/history:query`

Sample Request: [`https://api.shipbob.com/2.0/inventory/history:query?EventCategory=SpotChecking&EndDate=2025-03-03&StartDate=2025-03-01`](https://api.shipbob.com/2.0/inventory/history:query?EventCategory=SpotChecking\&EndDate=2025-03-03\&StartDate=2025-03-01)

### Query Parameters

| Query string key | Value    | Description                                                          |
| ---------------- | -------- | -------------------------------------------------------------------- |
| EventCategoryÂ    | string   | The category of an inventory event to return - see list below        |
| Limit            | integer  | The number of products to return in the response                     |
| Page             | integer  | The page number to return, if pagination is needed                   |
| StartDate        | datetime | The start time the query should return inventory history events for. |
| EndDate          | datetime | The end time the query should return inventory history events for.   |
