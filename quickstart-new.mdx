## 1. What is ShipBob API?

The ShipBob API helps you perform admin operations on the returns generated via ShipBob API in case there is an enterprise's integration need for their internal system programmatically.

## 2. Get the API key

ShipBob verifies a user's request by adding an as-api-key in the header.

To get your API key, visit API key, click Create an API key and follow the given instructions to generate your API key.

Button: Get API Key

## 3. Get your channel ID

## 4. Common scenarios

A few common scenarios to get started with the ShipBob API.&#x20;

### Create product

Make sure to pass the `shipbob_channel_id` in the header.

```
{
  "reference_id": "TShirtBlueM",
  "sku": "TShirtBlueM",
  "name": "Medium Blue T-Shirt"
}
```

<CodeGroup>
  ```json
  {
    "reference_id": "TShirtBlueM",
    "sku": "TShirtBlueM",
    "name": "Medium Blue T-Shirt"
  }
  ```
</CodeGroup>
